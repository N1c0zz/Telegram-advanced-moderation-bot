Informazioni sui Dati

Cosa contengono i file CSV:

messages.csv

Contiene tutti i messaggi processati dal bot con le seguenti informazioni:

  • Timestamp del messaggio
  • Testo del messaggio
  • ID utente e username
  • ID chat e nome gruppo
  • Stato di approvazione (SI/NO)
  • Se è una domanda (SI/NO)
  • Motivo di rifiuto (se applicable)

admin_messages.csv

Messaggi inviati da utenti amministratori (esenti da moderazione):

  • Timestamp del messaggio
  • Testo del messaggio
  • ID utente e username admin
  • ID chat e nome gruppo

banned_users.csv

Lista degli utenti bannati automaticamente dal sistema:

  • ID utente bannato
  • Timestamp del ban
  • Motivo del ban

Privacy e Sicurezza

  • • I dati sono crittografati durante il trasferimento
  • • I file contengono solo informazioni necessarie per la moderazione
  • • Gli ID utente sono pseudonimizzati quando possibile
  • • I backup vengono conservati secondo le policy di ritenzione

Note Importanti

  • • I file possono essere di grandi dimensioni a seconda dell'attività
  • • Il download potrebbe richiedere alcuni minuti per essere preparato
  • • I dati sono aggiornati in tempo reale
  • • Si consiglia di effettuare backup regolari
